# 🌱 Trait Prediction using Machine Learning in Plant Breeding

## 📌 Overview
This project applies machine learning techniques to predict important quantitative traits in cotton, including:

- Seed Cotton Yield  
- Lint Yield  
- Seed Yield  

The workflow integrates **data preprocessing, model training, evaluation, and visualization**, making it useful for plant breeding and crop improvement programs.

---

## 🎯 Objectives
- Predict yield-related traits using phenotypic data  
- Identify key traits influencing yield  
- Validate machine learning predictions with statistical relationships  
- Build a reproducible pipeline for research applications  

---

## 🧠 Methods Used
- Random Forest Regressor  
- Multi-output regression  
- Feature scaling (Standardization)  
- Train-test split validation  

### 📊 Evaluation Metrics
- R² Score  
- RMSE (Root Mean Square Error)  
- MAE (Mean Absolute Error)  

---

## 📊 Traits Included
- Plant Height  
- Sympodia  
- Monopodia  
- Number of Bolls  
- GOT (%)  
- Yield  
- Lint Yield  
- Seed Yield
